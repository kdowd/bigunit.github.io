Mediaworks Video On Demand B2B
  1. Modular vanilla Javascript
  2. SASS/SCSS for the styles
  3. Gulp for the build tasks
  4. Flexbox for the layout and filtering.

Responsive site built by myself for Mediaworks NZ. Its a business-to-business site. The aim is to show-off the creative output of the Mediaworks team and let clients and advertisers know the kind of thing they can expect if they buy a video deal from Mediaworks.

Regional Sales people use this site as a primary sales tool to sell pre-roll video across Medaiworks properties.

The layout is handled by flexbox and user filtering by javascript and flexbox together.

There is also drag and drop functionality on the preview page. This means we can drag an mp4 into the page and have it play in the video player - very useful for testing those AfterEffects builds in-situ.
